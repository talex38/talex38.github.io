About This Course

This page is built in part to demonstrate work for the Charlotte class ITIS3135 - Front-End Web Application Development, for sections run by Mr. von Briesen. You may find additional information about this course at the official course website.

This course seeks to provide students with relevant, hands-on experience in front-end web application development. Throughout the semester, we are building multiple websites using a variety of modern tools and techniques.

Tools Used

  • VS Code
  • Emmet within VS Code
  • SFTP via FileZilla
  • Git via GitHub Desktop and GitHub.com
  • Various Browser Developer Tools
  • The Accumulus Validator

Sites Being Built

The projects we will build (and which this page will link to) include:

  • A personal home portfolio
  • This course site
  • A design firm site (an aspirational design/consulting firm named after myself)
  • A "Crappy" page (intended to do all the bad things in web design, so as to avoid them everywhere else)
  • A whimsical product company
  • A student/colleague/peer management tool
  • A client project website
